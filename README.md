## About

This repository contains the source code for [qgn.app](https://qgn.app), the marketing website for the QGN screenshot tool.

## Tech Stack

- **Next.js 16** — React framework with App Router
- **React 19** — UI library
- **Tailwind CSS 4** — Utility-first styling
- **TypeScript** — Type safety
- **Lucide React** — Icon library

## Development

```bash
git clone https://github.com/Mopra/qgn.app.website.git
cd qgn.app.website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
